# Install shellpass

## Install prebuilt binaries via shell script

```sh
curl --proto '=https' --tlsv1.2 -LsSf https://github.com/tech0ne/shellpass/releases/latest/download/shellpass-installer.sh | sh
```

## Install prebuilt binaries via powershell script

```sh
powershell -ExecutionPolicy Bypass -c "irm https://github.com/tech0ne/shellpass/releases/latest/download/shellpass-installer.ps1 | iex"
```
